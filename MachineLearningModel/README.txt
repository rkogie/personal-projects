
--------------------------------------------------------------------------------------------------
Predictive Analysis Model  - PROJECT FORMULATION, PLANNING & IMPLEMENTATION
--------------------------------------------------------------------------------------------------

Objective -  Build a basic algorithm to predict 
		with relative accuracy, the chances of mortgage loan approval for prospective applicants.

Tasks     -  Model will analyze test features against the target variable.
		Model to parse through test data developed through training data and output the loan acceptance result

Assumptions	- 
			Total household income
			Credit history
			Loan amount requested
			Payment period
			Monthly payment structure
